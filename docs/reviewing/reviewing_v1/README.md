## Convergence des relecteurs

Quatre points sont soulevés indépendamment par au moins deux relecteurs :

1. **Encodage IFN-I défaillant (HDAC3 = STAT1)** — relecteurs 1, 2, 3.
2. **Distance Hamming sans null model, métrique mal calibrée** — relecteurs 1, 3.
3. **Le module AP1/p38 peut être un artefact topologique** — relecteurs 1, 2, 4.
4. **« 8/10 concordance clinique » trompeur** — relecteurs 2, 4.
5. **Pas de simulation effective des combinaisons (JAK + p38)** — relecteurs 1, 2, 4.
